# Baseera Roadmap

## Phase 1 – Blind-First MVP (0–6 months)

- PDF ingestion and structured text extraction
- Semantic chunking into learning units
- Concept breakdown and guided explanations
- Adaptive Q&A loop
- Voice-first interaction layer

## Phase 2 – Accessibility Expansion (6–12 months)

- Improved pedagogical structuring engine
- Multilingual support (Arabic-first, extensible)
- Beta release with real user feedback

## Phase 3 – Sign Language Support (Future Roadmap)

- AI-generated sign-language learning support
- Multimodal accessibility layer
- Research-driven development
