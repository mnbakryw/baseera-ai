# Architecture Overview

Baseera is built as a modular accessibility-first learning engine.

## Core Layers

1. Document Processing Layer  
   - PDF extraction  
   - Text cleaning and structuring  

2. Semantic Reasoning Layer  
   - Concept extraction  
   - Pedagogical restructuring  
   - Question generation  

3. Interaction Layer  
   - Voice-first tutoring loop  
   - Adaptive feedback  

4. Future Multimodal Layer  
   - Sign-language learning support
